# UrlShorter
Сокращатель ссылок  
Сервис для превращения длинных ссылок в короткие

## Стек технологий
1. ASP.NET (основа веб-сервиса)
2. PostreSQL (для хранения токенов)
3. Redis (для атомарного счетчика)
4. Docker (для развертывания веб-сервиса)
